# AI18_PROJECT3

AI 서비스 기획/개발 과정 미니프로젝트3 - 2_규칙기반 추천 시스템 (Streamlit)

## 포트폴리오 웹사이트

Streamlit을 사용하여 만든 개인 포트폴리오 웹사이트입니다.

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
streamlit run app.py
```

브라우저에서 자동으로 열리며, 기본적으로 `http://localhost:8501`에서 실행됩니다.

## 기능

- **홈**: 소개 및 프로필
- **프로젝트**: 진행한 프로젝트 목록 및 설명
- **기술 스택**: 사용 가능한 기술들
- **연락처**: 이메일, GitHub, LinkedIn 등 연락처 정보

## 기술 스택

- Streamlit
- Python
- HTML/CSS
